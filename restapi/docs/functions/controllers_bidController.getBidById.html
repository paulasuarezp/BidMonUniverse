Function getBidById

Recupera una puja específica por su identificador. Devuelve la puja si existe, de lo contrario, devuelve un mensaje de error.

Throws

404 - Si no se encuentra la puja con el ID proporcionado.

Throws

500 - Si se produce un error durante el proceso de recuperación de la puja.

  • getBidById(req, res): Promise<Response<any, Record<string, any>>>

  • Parameters

    • req: Request<ParamsDictionary, any, any, ParsedQs, Record<string, any>>

      El objeto de solicitud HTTP, que debe incluir el ID de la puja en req.params.id.

    • res: Response<any, Record<string, any>>

      El objeto de respuesta HTTP utilizado para enviar la puja recuperada o un mensaje de error.

    Returns Promise<Response<any, Record<string, any>>>

    • No retorna un valor directamente, pero envía una respuesta HTTP. Si la puja se encuentra, devuelve un estado 200 y la puja en formato JSON. Si la puja no se encuentra, devuelve un estado 404 con un mensaje de error. Si ocurre un error al recuperar la puja, se devuelve un estado 500 con un mensaje de error.
